Infinite Grid Pro is an unique responsive HTML5 multimedia infinite grid that can only be found on sale here, it runs on all major browsers and mobile devices like iPhone, iPad, IOS, Android and Windows desktop or mobile. Supports unlimited categories and each category can have unlimited content. When a thumbnail is pressed you can choose to do nothing, display multimedia content using our great revolution lightbox or to open a new webpage. The lightbox has support for image, video (mp4), audio (mp3), Vimeo, YouTube, iFrame, Google maps and flash. Infinite Grid Pro has high performance using OOP code and the latest CSS3 techniques, ready for mobile with swipe and pinch support, very flexible, easy to setup and extremely customizable.

The way Infinite Grid Pro is handling the thumbnails is as follows. Once the grid is initialized the first available thumbnails from the current category are added in order from left to right, after a drag or zoom is performed using a complex mathematical algorithm new thumbnails are added randomly, the new added thumbnails will remember their place in the grid no matter in which direction the grid is dragged and at the same time IGP is not allowing the same thumbnail to be repeated in it's close proximity.

Infinite Grid Pro main features:

+ Responsive layout and customizable color theme.

+ Mobile and desktop optimized (Android demo or Ipad demo).

+ Multiple display types, responsive, fixed, fullscreen, fluid width and after parent.

+ Support for multiple categories.

+ Optional drop down categories menu, this categories selector can be positioned on the left or right side of the grid canvas.

+ Custom press thumbnail action. When a thumbnail is pressed you can choose to do nothing, display multimedia content using our great revolution lightbox, or to open a new webpage.

+ Revolution lightbox used to display multimedia content. When a thumbnail is clicked the lightbox can be used to display image, video (mp4), audio (mp3), Vimeo, YouTube, iFrame, Google maps or flash.

+ Optional visited thumbnails, this allows thumbnails that were clicked or viewed with the ligthbox to have reduced opacity, this way the user can easily see which content was viewed.

+ Customizable drag, the grid drag direction can be set to horizontal, vertical, both or none.

+ Two types of thumbnail transition animation (opacity or motion).

+ Auto scroll, the grid can be set to auto scroll, the auto scroll direction and speed are customizable.

+ Optional grid zoom.

+ Customizable thumbnails interactivity (thumbnails can be set to do nothing when clicked).

+ Randomizable categories, each playlist can load and show thumbnails in random order or in normal order.

+ Optional thumbnails overlay, also a specific color can be set individually for each thumbnail overlay.

+ Optional thumbnails multimedia icon.

+ Optional help screen.

+ Two skins included along with the psd files.

+ Detailed documentation and sample files included.
